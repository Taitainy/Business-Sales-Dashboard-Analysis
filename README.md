# Business-Sales-Dashboard-Analysis
This project focuses on analyzing a retail business sales dataset to uncover key insights that drive revenue, customer behavior, and product performance. The dataset was cleaned, transformed, and visualized using Excel to create an interactive dashboard for decision-making.
## Objectives
- Analyze overall business performance
- Identify top-performing products and materials
- Evaluate sales distribution by product placement
- Understand customer purchasing patterns
- Highlight regional performance differences
